# required
Значение ДОЛЖНО быть массивом. Этот массив ДОЛЖЕН иметь хотя бы один элемент. Все элементы массива ДОЛЖНЫ быть уникальными строками.

Условие: верно, если объект содержит все свойства значения **"required"**.
