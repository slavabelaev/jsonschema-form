# default
На значение не накладывается никаких ограничений.

Это ключевое слово можно использовать для представления значения по умолчанию. РЕКОМЕНДУЕТСЯ, чтобы это значение было верным для связанной с ним схемой.

## Пример
```json
{
  "type": "boolean",
  "default": false
}
```
