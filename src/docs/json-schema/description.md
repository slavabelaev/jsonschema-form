# description
Значение ДОЛЖНО быть строкой.

Это ключевое слово является подробным описанием схемы.

## Пример
```json
{
  "description": "Идентификационный номер налогоплательщика",
  "pattern": "^([0-9]{10}|[0-9]{12})$"
}
```
